# Online-Learning-Transformers

## Instructions to run the code

### Required directory structure to save the results:

```

|   +-- Base/
|   |   +--Results/
|   |   |  +--res/
|   |   |  +--loss-curves//
|   +-- Data/
|   |   +--data1
|   |   +--data2
|   |   +--data3
|   |   ...

...
```
### Install the required packages:

```
pip install -r requirements.txt

```

### Run the command for KKD training

```
python main.py
```

Available arguments:
- `--t`: Dataset Type
- `--n`: Dataset Name
